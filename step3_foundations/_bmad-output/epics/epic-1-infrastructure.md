# Epic 1: Infrastructure et Setup Technique

## Vue d'ensemble
L'application est opérationnelle avec l'infrastructure de base permettant la gestion d'événements - setup Next.js 15+, PocketBase, design system warm pastel, et architecture technique complète.

## Objectif Business
Établir les fondations techniques robustes et modernes qui permettront le développement rapide et maintenable de toutes les fonctionnalités de gestion d'événements.

## Valeur Utilisateur
Les développeurs peuvent commencer à implémenter les fonctionnalités métier immédiatement avec une base technique solide, type-safe et performante.

## Exigences Couvertes
- Template de démarrage Next.js 15+ avec App Router
- Infrastructure PocketBase containerisée
- Design system warm modern pastel
- TypeScript strict avec architecture projet

## Stories Incluses
- Story 1.1: Initialisation Next.js avec App Router  
- Story 1.2: Configuration PocketBase et Collection Events
- Story 1.3: Setup Design System et shadcn/ui
- Story 1.4: Architecture TypeScript et Structure Projet

## Critères de Succès
- ✅ Application Next.js 15+ fonctionnelle avec TypeScript strict
- ✅ PocketBase opérationnel avec collection events configurée
- ✅ Design system warm pastel intégré avec shadcn/ui
- ✅ Structure projet et types TypeScript établis
- ✅ TanStack Query configuré pour data fetching
- ✅ Zéro erreur TypeScript, zéro usage d'`any`

## Dépendances
Aucune - Epic fondationnel

## Definition of Done
- Toutes les stories 1.1 à 1.4 complétées
- Application démarre sans erreur
- Tests de base passent
- Documentation technique de base créée
